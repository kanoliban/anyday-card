# AnyDayCard

A premium greeting card platform that combines AI-powered personalization with beautiful illustrated designs to create meaningful, one-of-a-kind cards for any occasion.

## How It Works

1. **Choose an occasion** — Birthday, anniversary, thank you, and more
2. **Answer questions** — Tell us about your recipient through a guided questionnaire
3. **Get personalized content** — AI generates unique messages based on your answers
4. **Select delivery** — Digital download, printable PDF, or print-and-ship

## Key Features

- **AI-Powered Personalization** — Guided questionnaire captures recipient details to generate truly personalized card messages
- **Premium Illustrated Designs** — Beautiful, curated card templates across multiple styles
- **Multi-Delivery Options** — Digital, printable PDF, or professionally printed and shipped
- **Template + AI Hybrid** — Beautiful designs enhanced with AI-generated personalized content

## Development

```bash
npm install
npm run dev
```

## Tech Stack

- Next.js 16
- React 18 + TypeScript
- Tailwind CSS
- Zustand (state management)
- Stripe (payments)
- OpenAI API (message generation)
